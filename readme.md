# 🌍 JeoGuessr

A fun geography guessing game inspired by GeoGuessr — built with pure HTML, CSS and JavaScript.

## How to Play

1. Open `index.html` in your browser
2. Click **Play** to start a 5-round game
3. Explore the Google Street View panorama to figure out where you are
4. Click on the mini-map to place your guess pin
5. Hit **Place your pin** to submit — the closer you are, the more points you earn (up to 5,000 per round)
6. After 5 rounds you'll see your total score and a summary of each round

## Features

- 🗺️ Embedded Google Street View for each location
- 📍 Interactive Leaflet map for placing guesses
- ⭐ Distance-based scoring (up to 5,000 points per round)
- 🏆 Final results with per-round breakdown and overview map
- 📱 Responsive design for desktop and mobile
- 🎨 Sleek dark UI with smooth animations

## Tech Stack

- **HTML5** — semantic structure
- **CSS3** — custom styling, gradients, animations, responsive layout
- **JavaScript** — game logic, Haversine distance formula, DOM manipulation
- **[Leaflet.js](https://leafletjs.com/)** — interactive map (loaded via CDN)
- **OpenStreetMap** — map tiles
- **Google Maps** — embedded Street View
